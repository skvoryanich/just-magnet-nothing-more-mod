# Just Magnet, nothing more

A minimalist mod for Minecraft Fabric 1.21.10 that adds **just magnets**. Nothing more.

## Philosophy

This mod does one thing and does it well: **magnets for item pickup**. No extra features, no bloat, no unnecessary complexity. Just magnets. That's our key feature - simplicity.

## What this mod adds

- **Magnet Core** - basic component for crafting magnets
- **Advanced Magnet Core** - improved version of the core
- **Magnet** - standard magnet for automatic item pickup
- **Advanced Magnet** - improved version with higher durability and pickup range

That's it. Nothing more.

## Features

- **Automatic item pickup** - magnets collect nearby items
- **Toggle on/off** - right-click to activate/deactivate your magnet
- **Server-side configuration** - adjust pickup radius and durability
- **Enchantment support** - works with Unbreaking and Mending
- **Recyclable** - smelt broken magnets back into cores to craft them again
- **Two tiers** - basic and advanced magnets for different needs

## Recycling

Broken or unwanted magnets can be smelted back into their cores, allowing you to craft new magnets. Nothing goes to waste!

## Configuration

Server-side configuration allows you to adjust:
- **Pickup radius** - how far the magnet can reach
- **Durability** - how long magnets last before breaking
